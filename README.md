# Portafolio de Enrique

## Antecedentes

Este es un portafolio que se creó para la evaluación del módulo 2 del bootcamp de Talento Digital. Fue un desarrollo rápido y eficiente. El objetivo es comenzar a utilizar tecnologías para la interacción jquery y javascript. Busqué desarrollar una interfaz simple y colores atractivos. Las lecciones las saqué luego de seguir las instrucciones de la tarea y contrastando con videos en youtube de portafolios de programadores.
